## 🖥️ Project Introduction
- 프로젝트 명 : 마당을 나온 드래곤 (Escape Dragon)
- 프로젝트 소개 : 오늘도 평화로운 하루…인줄 알았으나 눈을 씻고 찾아봐도 내가 기르던 용들이 모두 없어졌다! 너네한테 들인 용 사료값이 얼만데 반드시 찾아야 한다! 우리에서 사라진 용들을 찾으러 떠나는 흥미진진 대모험!
